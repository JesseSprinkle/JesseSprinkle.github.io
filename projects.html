The following projects were done for my course CSCE 489 - Computer Animation

An Implementation of Smoothed Particle Hydrodynamics (Webpage)

I implemented Smoothed Particle Hydrodynamics, a fluid simluation algorithm, as a final project for my Computer Animation course. To complete it I had to research, test, optimize, multithread, and tune parameters, and managed to get a reasonable simulation. Video can be found on the webpage. I also wrote a report on my method: I reset my computer before pushing the source code to Github, so I unfortunately don't have a copy of the code.
